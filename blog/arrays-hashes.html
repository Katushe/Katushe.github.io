Arrays and Hashes

DBC Phase 0 Week 4

January 2, 2016

Arrays and hashes are both used to store collections in Ruby, but they are not always interchangeable.

Arrays store collections of objects, and can hold any type of object-strings, integers, floats, you can even have an array of arrays. But you cannot mix types within a single array.Arrays are ordered, meaning that they have some sort of order, even if they’re not completely sorted from smallest to largest. Each element in an array has an integer index which tells you where it is located in the array. The index is how you access an element’s value. It is important to remember that in Ruby (and most languages) the first index in an array is 0, and thus the last index is not equal to the number of elements in the array, but rather the number of elements – 1.

There are a few ways to declare an array in Ruby:

array_name = Array.new
Will create a new empty array without a specified size.

array_name = Array.new(5)
Will create a new empty array that has a size/length of 5 elements.

colors = ["red","green", "blue"]
Will create a new array with some of the elements already specified.

To access the value of an element use:
array_name[index]

For example,
colors[0]
Using the above array will return "red".

Hashes on the other hand, hold collections of key-value pairs. Instead of accessing each element by an integer index, you explicitly assign every element a key and a value. Keys are then used to access the value, and thus must be unique. This means you cannot use the same key for different values within the same hash, otherwise the program would not know which value you want when you reference a key. You can however, assign the same value to multiple keys. Both the key and the value can be any type of object, and you can use a mix of different objects within a single hash.

There are a few ways to declare a hash in Ruby:

hash_name = Hash.new
Will create a new empty hash, the default value for any new key will be nil, unless otherwise specified.

hash_name = Hash.new(value)
Will create a new empty hash, but now whatever is in parenthesis will be the default value.

colors = {"r" => "red", "g" => "green", "b" => "blue"}
Will create a new hash with those elements specified, where r, g, b, are keys and red, blue, green are their corresponding values.

To access the value of an element use:
hash_name[key]

For example,
colors["r"]
Using the above hash will return "red".
