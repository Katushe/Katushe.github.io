The Box Model

DBC Phase 0 Week 3

December 20, 2015

Every element in an html document has a box around it. Inside this box is the element’s content, padding, border, and margins. This is referred to as the Box Model, which is pictured in the image above. The padding and margins are visible only as space between content and elements.

In the center of the box is the content area, which will contain the actual content of your element . This will be text, images, etc. The size will be determined by the size of your content, which you can either specify in your CSS or use the defaults set by your browser.

Next comes the padding, which creates an area around the content of the element. The padding is transparent and only serves to create space between the content and it’s border, but if you have a background color or background image applied to your content, it will affect the padding as well.

Then comes the border, which is an edge that goes around your content and it’s padding. This is considered the outside edge of the element. The border can also be transparent, but you can assign it a color separate from your content, and you can define a style for it, including but not limited to solid, dotted, dashed, and double.

Last is the margin, which creates an area around the border of the element. The margin is transparent and serves to create space between different elements on the page. It is not considered to be part of the element itself, and thus will not be affected by the background color of the content, and you cannot give it its own color.

While they both serve to create space, margins are used to make space between sibling elements that are next to/ above or below each other on the page and padding is used to create space between an element and its parent.


Both padding and margins can be set in multiple ways. And you can use different kinds of units, such as pixels, percents, or ems.


You can set the margins all around to the same value at once.

padding: 30px
margins:40px


You can set each side to a different value in one line. It goes in order of top, right, bottom, left.

padding: 30px 0 10px 0
margins:40px 50px 45px 20px


You can also set each side individually.

padding-top:5px
padding-right:15px
padding-bottom:20px
padding-left:10px
margin-top:10px
margin-right:20px
margin-bottom:30px
margin-left:50px
